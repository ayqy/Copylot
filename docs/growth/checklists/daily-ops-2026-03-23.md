# 7 天游程每日执行清单（D0-D6，2026-03-23 起）

目标：在产品已上线前提下，**每天只做少量确定动作**，持续拿到安装、激活与可复现反馈。

所用物料与表格：
- 渠道文案包：`docs/growth/publish-pack-2026-03-23.md`
- 指标记录表：`docs/growth/metrics-tracker-2026-03-23.md`
- 手动发布清单（D0）：`docs/growth/checklists/manual-posting-2026-03-23.md`
- 反馈处理清单：`docs/growth/checklists/feedback-triage-2026-03-23.md`

## 每天固定动作（不管发不发帖都要做）

1. 回复评论/私信/Issue（目标：24h 内首次回应）
2. 把新增反馈落盘到指标表「反馈记录」
3. 选 1 条最高频痛点，更新下一天的发帖角度（只改 1 件事）

## D0（2026-03-23）首发：通用痛点 + 3 卖点

发布（3 发）：
- X：`publish-pack` 4.1（EN）或 4.2（中文），`utm_campaign=twitter`
- LinkedIn：`publish-pack` 4.3（EN），`utm_campaign=linkedin`
- Reddit 或 HN（二选一）：`publish-pack` 4.4 或 4.5，`utm_campaign=reddit/hn`

收尾：
- 将 3 条“帖子 URL”写入：`docs/growth/metrics-tracker-2026-03-23.md`
- 记录当日曝光/点击/安装（可先填粗略值，收尾再补精确）

量化目标（D0）：
- 点击 ≥ 120；安装 ≥ 30；Activated ≥ 15；有效反馈 ≥ 5；Reviews ≥ 1
