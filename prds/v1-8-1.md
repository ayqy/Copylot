# V1-8-1 代码块清理误删修复（纯函数抽离 + 行级保守清理）

## 目标
将代码块“复制按钮文案”清理从全局替换改为行级、位置受限、强约束的保守清理，避免误删代码正文，使复制出来的代码可直接使用。

## 具体任务
1. 新增纯函数模块：`src/shared/code-block-cleaner.ts`
   - 导出：`cleanCodeBlockText(text: string): string`
   - 仅当某行 `trim()` 后完全等于候选按钮文案时才移除该整行
   - 默认仅清理首行/末行命中；允许清理首/末端连续段；严禁行内子串替换
   - 继续保留 Markdown 字符反转义逻辑，但仅作用于清理后的最终文本
2. `src/shared/content-processor.ts` 统一复用 `cleanCodeBlockText()`，确保所有 `pre/code` 路径一致

## 验收标准
- 不再出现“Copy/Clone 被全局替换导致代码缺失”
- 清理行为满足“行级 + 位置受限 + 强约束”
