# v1-100-2 问卷最短路径与漏斗证据

## 子目标
把 “入口点击 -> 问卷开始 -> 问卷提交” 固化为本地可导出的最小漏斗，并生成可审计证据包。

## 具体任务
- 新增纯函数模块：`src/shared/pro-intent-funnel-v1-100.ts`
  - 统计 `upgradeEntryClicks / formStarts / formSubmits`
  - 计算 `formStartRate / intentSubmitRate`
  - 导出稳定 CSV
- Options 隐私页新增两个导出入口：
  - `#download-pro-intent-v1-100-summary-json`
  - `#download-pro-intent-v1-100-summary-csv`
- 新增证据生成脚本：
  - `scripts/build-pro-intent-v1-100-evidence.ts`
  - 输出到 `docs/evidence/v1-100/`
    - `intent-funnel-v1-100.csv`
    - `intent-funnel-summary-v1-100.json`
    - `intent-sample-audit-v1-100.json`
    - `index.md`
- 样本说明必须如实记录：
  - runtime telemetry 只记录 `popup|options`
  - 官网 / 商店来源通过 UTM 与 CTA content 桥接进入同口径
  - 不虚构第三类 runtime source

## 验收标准
- `docs/evidence/v1-100/` 可直接用于复盘
- 三类来源样本至少各 1 条
- totals 与样本审计一致
