# Roadmap 状态

## 当前阶段

- 阶段名称：S2 / G2 `第二次复用与口碑循环`
- 阶段目标：把第一次成功复制推进成“愿意再次打开 -> 愿意继续复用 -> 愿意分享或评价”的低打扰增长循环。
- 当前判断：`prds/v4.md` 已把本阶段主线收敛为“第二次打开理由 + 高频场景稳定性 + 价值验证后的口碑触发”，当前最小可交付增量已经先落到第二次打开理由，接下来应围绕真实复用后的稳定性与口碑时机继续推进。
- 当前交付边界：继续不做表单、问卷、候补、支付或收款承接；本阶段所有动作都要直接服务真实复用与后续口碑转化质量。

## 下一步最重要的 3 件事（收入优先）

1. 针对代码块场景补一轮“复制后无需返工”的稳定性专项，优先修掉影响复用的缩进、行号和噪声残留问题，并保留可回归证据。
2. 把分享与评价提示绑定到“已完成至少 2 次成功复制”之后，并保留可导出证据，让口碑动作只发生在价值已被验证的用户上。
3. 继续压实 Prompt 默认命中与追加模式协同，提升第二次打开后的真实复用频率，而不是只增加入口曝光。
