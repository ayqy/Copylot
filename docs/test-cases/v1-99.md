步骤：
1. 检查 `conversionEntries` 中 v1-99 新增样本数是否为 3 且覆盖三入口。
2. 检查每条新增样本 `postUrl` 非占位值。
3. 检查每条新增样本包含 `postUrl/targetUrl/campaign/source/medium/intentSignal/clicks/installs/proIntentSignals/evidencePath`。
4. 证据落盘：`docs/evidence/v1-99/case-a-sample-integrity.json`。

预期：
- 结果 PASS；
- 新增样本三入口齐全；
- 至少 2 条样本满足 `clicks > 0` 且（`installs > 0` 或 `proIntentSignals > 0`）。
