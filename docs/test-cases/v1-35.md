## 用例 A（整页转换：正文优先）
前置：
- 使用包含 `main` 或 `article` 的“文章页”（有 header/nav/aside 噪音区域 + 正文区域）。

步骤：
1. 确保页面无任何选区（点击空白处取消选区）。
2. 通过右键菜单或扩展入口触发 Convert Page（消息类型 `CONVERT_PAGE`）。
3. 粘贴剪贴板内容进行观察。

期望：
- 结果仅包含正文区域内容。
- 不包含 header/nav/aside/footer 等区域中的干扰文本。
