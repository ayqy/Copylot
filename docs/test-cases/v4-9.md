## 用例 A：Pro Tab 出现领先路线回写包区块

步骤：

1. 打开 `src/options/options.html#pro`。
2. 检查 `#pro-route-writeback-panel`、`#copy-pro-route-validation-writeback-summary`、`#download-pro-route-validation-writeback-json`。

期望：

- 新区块可见。
- 两个动作按钮存在且默认可点击。
- 隐私提示明确说明不包含页面正文、复制内容、URL、标题或联系方式。

## 用例 B：复制出的回写包包含三段对外文案

步骤：

1. 在 `Options -> Pro` 点击“复制领先路线回写包”。
2. 读取最近一次剪贴板文本。

期望：

- 文本包含 `V4-9 领先路线回写包`。
- 文本包含当前领先路线标题，以及 `recent_7d total_signals` / `signal_gap` / `route_opened` / `total_copies`。
- 文本同时包含路线页、商店说明、汇总三段内容。

## 用例 C：下载出的 JSON 可复核

步骤：

1. 点击“下载领先路线回写 JSON”。
2. 打开生成文件并检查关键字段。

期望：

- 文件名符合 `copylot-pro-route-validation-writeback-v4-9.json`。
- 文件内包含 `leadingTrackId`、`routePage`、`storeCopy`、`summary`。
- 若匿名数据关闭，结果仍可导出，但必须回到无领先路线的兜底文案。
