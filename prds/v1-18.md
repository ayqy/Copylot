## 具体任务
1、扩展 `copilot_growth_stats` 结构以覆盖漏斗里程碑（仅时间戳/计数/枚举；可规范化、可回滚）
  - 修改 `src/shared/growth-stats.ts` 的 `GrowthStats` 与规范化逻辑，新增字段（全部可选；类型必须为毫秒时间戳 `number`）：
    - `firstPopupOpenedAt?: number`
    - `firstSuccessfulCopyAt?: number`
    - `lastSuccessfulCopyAt?: number`
    - `firstPromptUsedAt?: number`
    - `reusedWithin7DaysAt?: number`（首次成功复制后 7 天内出现的“第二次成功复制”时间；仅写入一次）
  - 规范化要求（必须可单测、可回归）：
    - 非法值（非 `number` / `<=0` / `NaN`）一律丢弃为 `undefined`
    - 兼容旧数据：老用户升级后不报错；必要时自动回写“规范化后的结构”
  - 约束：
    - 严禁加入任何文本字段、URL、标题、复制内容
    - 字段总量控制在“最小可交付”，避免反馈模板 URL 过长

2、将漏斗里程碑写入增长统计（只在成功路径写入；失败不写；不影响主流程）
  - Popup 首次打开：
    - 在 `src/popup/popup.ts` 的 `initialize()` 中（记录 `popup_opened` 同一初始化路径即可）写入 `firstPopupOpenedAt`（仅当该字段不存在时写入）
    - 建议在 `src/shared/growth-stats.ts` 新增专用函数（例如 `markFirstPopupOpened(...)`）以保持写入逻辑集中
  - 成功复制（复用既有链路，避免引入新依赖/新权限）：
    - 复用 `src/content/content.ts` -> `chrome.runtime.sendMessage(type='growth-stats-increment-successful-copy')` -> `src/background.ts` -> `src/shared/growth-stats.ts`
    - 在 `incrementSuccessfulCopyCount(...)` 内同时维护：
      - `firstSuccessfulCopyAt`：若不存在则写入本次时间
      - `lastSuccessfulCopyAt`：每次成功复制都更新为本次时间
      - `reusedWithin7DaysAt`：当 `successfulCopyCount` 从 `1` 变为 `2` 且 `now - firstSuccessfulCopyAt <= 7d` 时写入（仅写一次）
  - Prompt 使用（以“Prompt 触发且复制成功”为准）：
    - 在 `src/content/content.ts` 现有 `recordTelemetryEvent('prompt_used')` 的成功路径上，扩展 `reportSuccessfulCopy()` 支持透传 `isPromptUsed=true`（或等价标记）到后台
    - 后台在同一次增长统计更新中写入 `firstPromptUsedAt`（仅当字段不存在时写入）
  - 容错与约束：
    - 任何 storage 写入失败不得阻断复制；必须 try/catch 并静默降级（仅 `console.warn`）
    - 不新增任何弹窗/提示；仅补齐本地可观测数据

3、补齐单测与测试用例文档（可执行、可回归、可发布）
  - 单测（最小但必须）：
    - 在 `src/shared/growth-stats.ts` 抽出纯函数（例如：`applySuccessfulCopyToGrowthStats(stats, { now, isPromptUsed })`），并在 `scripts/unit-tests.ts` 增加断言覆盖：
      - 首次/最近成功复制时间戳写入规则（首次只写一次；最近每次更新）
      - `isPromptUsed` 仅影响 `firstPromptUsedAt` 且只写一次
      - `reusedWithin7DaysAt` 满足条件才写入，且只写一次
      - 新增字段的非法值在 normalize 后被清理
  - 测试用例文档：
    - 新增并维护 `docs/test-cases/v1-18.md`，覆盖（至少）：
      - 清空/重置后首次打开 Popup：`firstPopupOpenedAt` 写入且不被覆盖
      - 首次成功复制：`firstSuccessfulCopyAt/lastSuccessfulCopyAt/successfulCopyCount` 正确
      - 二次成功复制：模拟在 7 天内触发时 `reusedWithin7DaysAt` 写入；超过 7 天不写入
      - Prompt 复制成功：`firstPromptUsedAt` 写入；普通复制不写入
      - 隐私校验：`copilot_growth_stats` 仅含时间戳/计数/枚举；不含网页内容、URL、标题、复制内容
      - 自动化回归：记录一次 `bash scripts/test.sh` 结论
  - 验收前完成回归：确保不影响既有评价引导触发、匿名事件日志、反馈模板、Pro/WOM、Prompt 管理与排序等功能，达到可发布状态
