# Roadmap 状态

## 当前阶段

- 阶段名称：S2 / G2 `第二次复用与口碑循环`
- 阶段目标：把第一次成功复制推进成“愿意再次打开 -> 愿意继续复用 -> 愿意分享或评价”的低打扰增长循环。
- 当前判断：v1-111 已把 Popup 首屏、商店说明、截图顺序、官网承接规范、安装指引与首次成功审计口径对齐，安装后的第一条主路径已经能稳定指向第一次干净复制。
- 当前交付边界：继续不做表单、问卷、候补、支付或收款承接，下一阶段所有动作都只服务复用与口碑。

## 下一步最重要的 3 件事（收入优先）

1. 把首次成功后的第二步固定成可复用动作：在 Popup / Onboarding 中强化快捷 Prompt 槽位或追加模式，形成明确的第二次打开理由。
2. 针对长文、表格、代码块三类高频场景补一轮“复制后无需返工”的稳定性专项，优先消灭影响复用的边角失败。
3. 把分享与评价提示绑定到“已完成至少 2 次成功复制”之后，并保留可导出证据，让口碑动作只发生在价值已被验证的用户上。
