# v1-100-1 三入口 CTA 归因对齐

## 子目标
让 Popup / Options / 外部落地页导入的 Pro 入口都写入同一套可审计归因字段，至少覆盖 `source / medium / content / campaign`。

## 具体任务
- 新增归因纯函数：`src/shared/pro-intent-attribution.ts`
  - 固化 `source=popup|options`
  - 固化 `content` 枚举：
    - `popup_upgrade_cta`
    - `popup_waitlist_cta`
    - `popup_survey_cta`
    - `options_waitlist_cta`
    - `options_survey_cta`
    - `options_survey_copy_open`
- 扩展 `src/shared/external-links.ts`
  - `buildOfficialSiteUrl/buildProWaitlistUrl` 支持 `utm_content`
- 扩展 `src/shared/telemetry.ts`
  - 新增 `pro_intent_form_start / pro_intent_form_submit`
  - 为 `pro_entry_opened / pro_waitlist_opened / pro_waitlist_copied / pro_waitlist_survey_copied` 放开 `medium/content/campaign`
- 对齐入口行为：
  - `src/popup/popup.ts`
    - Upgrade CTA 记录 `pro_entry_opened`
    - Popup 问卷 CTA 记录 `pro_entry_opened + pro_intent_form_start`
    - Popup waitlist CTA 记录 `pro_entry_opened + pro_waitlist_opened`
  - `src/options/options.ts`
    - Pro tab 打开、问卷 copy / copy-open、waitlist open 都使用统一 attribution

## 验收标准
- 所有 Pro 入口都能导出合法 `source / medium / content / campaign`
- 非法 source 不会污染 `campaign`
- 外部链接会带 `utm_content`
