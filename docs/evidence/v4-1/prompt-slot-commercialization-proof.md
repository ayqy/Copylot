# v4-1 快捷 Prompt 槽位商业化证据口径

## 本轮可验证证据

1. 快捷 Prompt 槽位曝光
   - 指标：`quickPromptSlotShownCount`
   - 价值：说明首次成功后，用户已经看到第二次打开的明确理由。

2. 快捷 Prompt 槽位点击
   - 指标：`quickPromptSlotClickedCount`
   - 价值：说明用户不只是看到了入口，而是愿意尝试复用路径。

3. 快捷 Prompt 槽位使用 + 第二次 `copy_success`
   - 指标：
     - `quickPromptSlotUsedCount`
     - `secondSuccessfulCopyAt`
     - `successfulCopyCount >= 2`
   - 价值：说明“复制网页内容 + 套用 Prompt + 粘贴到 AI”已经成为真实动作，而不是静态按钮。

## 为什么这能支撑商业化判断

- 如果曝光有、点击低：
  - 说明入口被看见了，但复用理由不够强，仍需打磨默认 Prompt 或文案。
- 如果点击有、第二次成功低：
  - 说明入口可信，但 Prompt 内容、复制结果或页面稳定性仍是阻塞点。
- 如果点击与第二次成功同时成立：
  - 说明首次成功已经开始转成真实复用，这会直接提升：
    - 留存与再次打开概率
    - 用户愿意分享“这是个能持续用的工具”的可信度
    - 后续再评估高质量分享、评价、Pro 路线说明时的转化基础

## 本轮结论

- 最关键的商业化前置信号不是“看到了多少按钮”，而是“首次成功后是否真的再次复制成功”。
- 因此本轮把快捷 Prompt 槽位曝光、点击、使用与第二次 `copy_success` 固定为同一条可审计证据链，足以支撑后续判断真实复用、分享质量与可转化增长。
