# 30 秒完成第一次干净复制

网页复制真正麻烦的地方，不是“复制不到”，而是复制出来还要自己清理。Copylot 把安装后的第一件事收敛成同一条路径：先试一个长文、表格或代码块，在 30 秒内复制出可以直接粘贴到 AI 或文档的干净结果。

## 前三个卖点

- 长文：去掉导航、广告和杂项说明，直接得到干净 Markdown / 纯文本。
- 表格：一键转成 Markdown / CSV，粘贴后结构仍然清楚。
- 代码块：保留缩进与空行，尽量去掉行号和复制按钮噪声。

## 快速上手（30 秒）

1. 从 Chrome Web Store 安装并固定 Copylot。
2. 打开一个长文、表格或代码块，点击 Popup 里的“复制给 AI”，或者使用你选定的单击 / 双击 / 悬停方式触发复制。
3. 把结果粘贴到 ChatGPT、Claude、Notion 或任意文档里，确认第一次干净复制已经完成。

## 推荐截图顺序（与官网首页一致）

1. Popup 首屏：`30 秒完成第一次干净复制`
2. 长文复制：去广告和导航后的干净 Markdown
3. 表格复制：Markdown / CSV 双场景
4. 代码块复制：保留缩进并去噪
5. 粘贴结果：把第一次结果贴到 AI 或文档

## 为什么先做第一次干净复制

- 安装后的首要目标不是先调设置，而是尽快验证“复制出来就能用”。
- 长文、表格、代码块三类场景已经足够覆盖第一次价值感知。
- 第一次成功后，再逐步使用追加模式、Prompt 快捷位和更多复用工作流。

## 核心功能

- 智能区块复制：正文优先，输出干净 Markdown / 纯文本。
- 表格转 Markdown / CSV：适合直接贴进文档、表格工具和 AI 对话。
- 代码块清理：尽量保留可运行结构，减少复制后返工。
- 追加模式：按住 Shift 连续收集多段内容，再一次粘贴。
- Prompt 快捷位：把常用指令和复制内容拼成一个稳定工作流。

## 隐私与权限

- 默认本地处理：提取、清理、格式化都在你的设备上完成。
- 不收集/不上传复制内容：不会把网页正文、代码或表格内容发到服务端。
- “匿名使用数据”开关默认关闭；开启后也只记录本地匿名事件，不包含复制内容。
- 最小权限：`storage`、`clipboardWrite`、`contextMenus`。
