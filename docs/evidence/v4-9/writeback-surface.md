# v4-9 领先路线回写入口

## 界面位置

- `Options -> Pro`
- 区块：`#pro-route-writeback-panel`
- 按钮：
  - `#copy-pro-route-validation-writeback-summary`
  - `#download-pro-route-validation-writeback-json`

## 输出边界

- 只使用本地匿名聚合信号。
- 不包含网页正文、复制内容、URL、标题或联系方式。
- 回写文案必须保持“路线验证中”口径，不得暗示支付、订阅或收费能力已经上线。

## 预期用途

- 路线页 headline / proof / boundary
- 商店说明 shortDescription / bullet / boundary
- 下一轮汇总 judgement / nextStep
