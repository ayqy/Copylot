💡 如何使用您的Prompt

1
选择文本：在任意网页中选择您想要处理的文本内容
2
右键菜单：右键点击选中的文本，在菜单中找到"Magic Copy with Prompt"
3
选择Prompt：从子菜单中选择您刚才创建的Prompt模板
4
自动复制：处理后的文本将自动复制到剪贴板，可直接粘贴到AI聊天界面

Copylot Pro（候补）

不打扰、不强引导；先验证意向，再决定是否投入实现。

Pro 会聚焦“更强工作流 / 更省时间”，在不破坏免费核心体验的前提下提供可选增强。

Free（永久免费）

以下为已发货的核心能力清单，并承诺长期免费（不做付费墙）。

  • 智能区块复制：单击/双击提取正文，复制为 Markdown / 纯文本
  • 代码块悬停复制 + 保守清理（缩进/空行、行号边界、Copy 文案整行移除）
  • 追加模式（Shift 追加合并，多段内容用分隔符整理）
  • 表格一键转 Markdown / CSV
  • Prompt 管理器（模板/分类/同步）与一键套用
  • 可配置默认 Chat 服务与复制后自动打开（基于本地设置）

Pro（规划中，未上线）

以下仅为候补/规划方向（Not shipped yet），不代表已提供。

  • 高级页面清洗（阅读模式 + 去广告/评论区/推荐区规则）
  • 批量采集与整理（跨页面收集、命名、导出、历史检索）
  • Prompt Pack 高级库/团队共享（后端能力后置）
  • 面向笔记工具的导出能力（Obsidian/Notion 结构化模板）
