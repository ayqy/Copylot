# V1-34-1 拆分：Prompt 作用域一致性修复（悬浮按钮 Prompt 优先选区）+ 用例/文档闭环

## 范围（MVP）
在不新增任何扩展权限、不改变既有复制/Prompt/引导/口碑/商业化功能主逻辑的前提下，修复悬浮按钮 Prompt UX 不一致：

- 当前块内存在有效选区：Prompt 必须优先作用于选区
  - 表格内选区：优先整表（table ancestor）
  - 非表格选区：优先精确选区 fragment（保留内联结构）
- 无有效选区 / 选区不在当前块：保持既有行为（仍作用于当前块，不被其它选区劫持）
- 右键菜单 Prompt 路径行为保持不变
- 统计与隐私无回归（本地匿名事件日志可审计；开关关闭清空且不记录）
