# V1-20 反馈模板补齐漏斗摘要：Issue 模板附带 GrowthFunnelSummary 测试用例

## 目标
验证 V1-20 达到可发布：
- Popup 与 Options 的反馈入口均可打开 GitHub `issues/new`，且 body 预填包含 5 块诊断信息：
  - 环境信息
  - `copilot_settings`
  - `copilot_growth_stats`
  - `copilot_growth_funnel_summary`
  - `copilot_telemetry_events`
- `copilot_growth_funnel_summary` 严格隐私合规：仅包含时间戳/计数/布尔/耗时（毫秒）等标量派生信息；不包含网页内容、URL、标题、复制内容
- 当“匿名使用数据”关闭时：`copilot_telemetry_events` 为 `[]`；其余 4 块仍存在；打开反馈页不报错
- `bash scripts/test.sh` 全量通过

## 预置条件
- 已在 Chrome/Chromium 加载本地扩展（开发包或生产包均可）
- 能打开 Popup 与 Options 页面：
  - Popup：点击扩展图标打开 Popup（`src/popup/popup.html`）
  - Options：右键扩展 -> 选项，或 `chrome.runtime.openOptionsPage()`（`src/options/options.html`）

## 用例 A（Popup 入口：反馈链接带 5 块预填信息）
步骤：
1. 打开 Popup。
2. 点击反馈入口 `#feedback-link`（页面会打开 GitHub `issues/new`）。
3. 在 GitHub 新建 Issue 页面中，检查 body 预填内容（或直接查看 URL 的 `body` query）。

期望：
- 成功打开 GitHub `issues/new`。
- body 中包含并按顺序出现以下 5 块（标题文本允许中英文差异，但关键标识必须存在）：
  - `copilot_settings`
  - `copilot_growth_stats`
  - `copilot_growth_funnel_summary`
  - `copilot_telemetry_events`
- `copilot_growth_funnel_summary` 段落为 JSON code block（可解析 JSON）。

## 用例 B（Options 入口：行为与 Popup 一致）
步骤：
1. 打开 Options 页面。
2. 找到 WOM 反馈入口 `#wom-feedback-open` 并点击。
3. 在 GitHub 新建 Issue 页面中检查 body。

期望：
- 行为与用例 A 一致：可打开 `issues/new`，且 body 预填包含相同 5 块诊断信息（含 `copilot_growth_funnel_summary`）。

## 用例 C（隐私校验：漏斗摘要仅标量信息）
步骤：
1. 按用例 A 或 B 打开 GitHub 新建 Issue 页面。
2. 定位到 `copilot_growth_funnel_summary` 的 JSON code block。
3. 检查所有字段类型与内容：
   - 允许：时间戳（number）、计数（number）、布尔（boolean）、毫秒耗时（number）
   - 禁止：任何网页内容、URL、标题、复制内容、用户输入文本等（例如出现 `http`、长文本段落、HTML 片段）

期望：
- `copilot_growth_funnel_summary` 不包含网页/复制内容、URL、标题或用户输入文本字段。

## 用例 D（匿名使用数据关闭：telemetry events 为 []，其余块仍存在）
步骤：
1. 打开 Options -> 隐私与可观测性，将“匿名使用数据”开关关闭（`#anonymous-usage-data-switch` 置为 off）。
2. 再次按用例 A（Popup）或用例 B（Options）打开反馈页面。
3. 检查 `copilot_telemetry_events` 段落内容。

期望：
- `copilot_telemetry_events` 的 JSON code block 内容为 `[]`。
- `copilot_settings` / `copilot_growth_stats` / `copilot_growth_funnel_summary` / 环境信息仍存在。
- 打开反馈页过程不报错（允许 `console.warn` 但不应阻断打开）。
