# 官网落地页取证资产（Landing Page Evidence）

用途：当外网恢复可达后，把官网落地页的**真实 Hero/首屏文案与截图**落盘到仓库，作为渠道文案对齐与增长复盘的可审计依据。

## 建议目录结构（按日期）

```text
docs/growth/assets/landing/2026-03-23/
  hero-copy.md
  hero.png
  section-1.png
  section-2.png
```

## 建议最小取证清单（每次更新至少做一次）

1. `hero-copy.md`
   - 标题（Headline）
   - 副标题（Subheadline）
   - 主 CTA 按钮文案
   - 首屏 3 个卖点（如有）
2. 截图至少 2 张
   - `hero.png`：首屏（含标题/副标题/CTA）
   - `section-1.png`：任意 1 个功能区块/场景区块

> 说明：若可用，优先用浏览器自动化（Playwright）抓取与截图；否则允许手动截图，但必须落盘到本目录，避免口径漂移。
