# V1-100 Pro 意向转化最小增量测试用例

## 目标

验证 `prds/v1-100.md` 的三项任务达成：
- 三入口 CTA 与 attribution 字段统一；
- 问卷最短路径可达且可记录 `form_start/form_submit`；
- v1-100 漏斗 JSON / CSV 与离线证据目录可导出、可复盘。

## 前置条件

- 代码模块：
  - `src/shared/pro-intent-attribution.ts`
  - `src/shared/pro-intent-funnel-v1-100.ts`
- 证据目录：`docs/evidence/v1-100/`
- 执行记录：`docs/growth/executions/v1-100-pro-intent.md`

## 用例 A：三入口 CTA 与参数一致性

步骤：
1. 在 Popup 点击 `升级 Pro`。
2. 在 Popup 点击 `Pro 问卷（1 分钟）`。
3. 在 Popup 点击 `加入候补名单`。
4. 在 Options -> Pro 点击 `加入候补名单`。
5. 复核 telemetry 与外链参数。

预期：
- `pro_entry_opened` 带合法 `source/medium/content/campaign`
- Popup waitlist URL 包含 `utm_content=popup_waitlist_cta`
- Popup survey copy-open URL 包含 `utm_content=popup_survey_cta`
- Options waitlist URL 带 `utm_content=options_waitlist_cta`

## 用例 B：问卷最短路径完整性

步骤：
1. 从 Popup 问卷入口打开 `options.html?pro_survey_source=popup#pro-waitlist-survey`。
2. 填写问卷并点击 `复制并打开候补链接`。
3. 复核事件日志中的 `pro_intent_form_start` 与 `pro_intent_form_submit`。
4. 验证匿名开关关闭时，提示条可见，点击 `开启并计入样本` 后提示消失。

预期：
- Popup deep-link 场景 `form_start/form_submit` 均归因到 `popup_survey_cta`
- 问卷样本提示可一键开启匿名使用数据
- 复制成功后 `pro_waitlist_opened` 归因不漂移

## 用例 C：商业化漏斗证据导出

步骤：
1. 在隐私页展开 `#pro-funnel-panel`。
2. 点击下载：
   - `#download-pro-intent-v1-100-summary-json`
   - `#download-pro-intent-v1-100-summary-csv`
3. 运行：
   - `./node_modules/.bin/ts-node scripts/build-pro-intent-v1-100-evidence.ts`
4. 复核 `docs/evidence/v1-100/`。

预期：
- JSON 文件名：`intent-funnel-summary-v1-100.json`
- CSV 文件名：`intent-funnel-v1-100.csv`
- 证据目录存在：
  - `intent-funnel-v1-100.csv`
  - `intent-funnel-summary-v1-100.json`
  - `intent-sample-audit-v1-100.json`
  - `index.md`
