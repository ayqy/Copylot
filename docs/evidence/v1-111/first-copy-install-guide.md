# v1-111 第一次干净复制安装指引

## 目标

安装完成后，不先讲复杂配置，只帮助用户最快完成一次可验证的干净复制。

## 三步完成

1. 安装并固定 Copylot
   - 从 Chrome Web Store 安装扩展。
   - 固定到浏览器工具栏，确保第一次打开就能看到 Popup 首屏。
2. 先选一个最容易验证的对象
   - 长文：新闻、博客、帮助文档正文。
   - 表格：价格表、对比表、统计表。
   - 代码块：技术文章或文档里的示例代码。
3. 触发复制并立刻粘贴验证
   - 打开 Popup，点击“复制给 AI”，或者使用你当前启用的单击 / 双击 / 悬停方式。
   - 立刻把结果粘贴到 ChatGPT、Claude、Notion、飞书文档或任意文本编辑器。

## 你应该看到的结果

| 场景 | 预期结果 |
| --- | --- |
| 长文 | 导航、广告、页脚等噪声减少，正文更适合直接贴给 AI |
| 表格 | 结构仍然整齐，可继续贴到表格工具或文档 |
| 代码块 | 缩进与空行保留，复制按钮文案和部分行号噪声被清理 |

## 如果第一次没有成功

1. 先回到 Popup 首屏，确认“启用智能复制”已打开。
2. 优先尝试更标准的长文正文、规则表格或代码块，而不是整个页面最外层容器。
3. 再次复制后立即粘贴验证，不要先做其他设置。

## 之后再做什么

- 需要跨段收集时再打开 Append Mode。
- 需要把指令与复制内容组合时，再去绑定 Prompt 快捷位。
