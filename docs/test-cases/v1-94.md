# V1-94 Pro 意向样本量提升：问卷区“计入样本”提示 + 一键开启匿名使用数据 + 证据回归（测试用例）

## 目标
验证子 PRD `prds/v1-94.md` 的验收点：在不引入支付/订阅、不新增权限、不上传用户复制内容的前提下，把「Pro 候补问卷」的样本可计入口径做成可执行闭环：
- 匿名开关 OFF 时，`#pro-waitlist-survey` 明确提示“不计入样本”，并支持一键开启（可逆；关闭会清空本地匿名事件日志）
- 匿名开关 ON 后，问卷复制动作可稳定计入 `survey_intent`，并可通过 v1-90 证据包导出与漏斗摘要复核
- 漏斗口径新增 `survey_copied_per_prompt_shown` 且在 UI/证据包中可复核（分母为 0 时为 `null`）

本轮验收对象（固定路径）：

- 自动化回归入口：`bash scripts/test.sh`

## 用例 A：匿名开关 OFF -> 问卷区提示“不计入样本” + 一键开启按钮

步骤：
1. 打开 Options，进入「隐私与可观测性」页，关闭「匿名使用数据」开关（关闭会清空本地匿名事件日志）。
2. 进入「Copylot Pro（候补）」页，滚动到 `#pro-waitlist-survey`。
3. 观察问卷区的“计入样本”提示与按钮。

期望：
- 开关 OFF 时：
  - 问卷区出现提示：“未开启匿名使用数据：本次填写/复制不会计入 `survey_intent` …”
  - 存在按钮：“开启并计入样本”

---

## 用例 B：点击“一键开启并计入样本” -> 开关 ON + 提示隐藏 + 面板刷新

步骤：
1. 在用例 A 的状态下，点击“开启并计入样本”按钮。
2. 切换到「隐私与可观测性」页确认开关已勾选。
3. 返回 `#pro-waitlist-survey` 观察提示是否隐藏。
4. 打开「Pro 意向漏斗摘要」面板，点击“刷新”，确认未出现旧口径残留提示（例如 disabled notice）。

期望：
- 开关变为 ON（可在隐私页看到已勾选）。
- 问卷区提示与按钮隐藏（避免噪音）。
- Pro 漏斗摘要面板可正常刷新。

---

## 用例 C：匿名开关 ON -> 产生样本 + 导出证据包并复核口径

步骤：
1. 保持匿名开关 ON（可用用例 B 一键开启）。
2. 在 `#pro-waitlist-survey` 填写任意字段（可为空），点击「复制问卷（含环境信息）」或「复制并打开候补」。
3. 在 Options -> 隐私与可观测性：
   - 展开「Pro 意向漏斗摘要」面板，确认漏斗字段存在
   - 点击「下载 Pro 意向跑数证据包（JSON）」并下载文件
4. 打开导出的 JSON，复核关键字段与隐私红线。

期望：
- UI 侧可观察：
  - `pro_waitlist_survey_copied >= 1`
  - `survey_copied_per_prompt_shown` 存在（分母为 0 时为 `null`）
- 证据包侧可复核（v1-90 导出）：
  - `proFunnelSummary.*.rates.survey_copied_per_prompt_shown` 存在
  - `proWaitlistSurveyIntentDistribution.survey_intent >= 1`
- 隐私红线：证据包不包含 URL/标题/网页内容/复制内容/联系方式明文。
