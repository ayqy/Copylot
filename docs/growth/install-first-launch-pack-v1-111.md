# v1-111 安装后第一次干净复制分发包

## 目标

把安装承接统一成一条最短路径：先让用户安装，再在第一次打开后立刻完成一次长文、表格或代码块的干净复制。

## 固定官方入口

- 官网首页：`https://copy.useai.online/`
- 商店安装页：`https://chromewebstore.google.com/detail/ai-copilot-%E2%80%93-magiccopy/ehfglnbhoefcdedpkcdnainiifpflbic`
- 隐私说明页：`https://copy.useai.online/privacy`
- Pro 路线页：`https://copy.useai.online/pricing`

## 推荐发送顺序

1. 第一次接触 Copylot 的用户：
   先发 Chrome Web Store 安装页，目标是尽快完成安装并进入第一次干净复制。
2. 用户想知道安装后先做什么：
   直接发送 [docs/evidence/v1-111/first-copy-install-guide.md](/Users/pocket/Documents/project/Copylot/docs/evidence/v1-111/first-copy-install-guide.md) 的三步安装指引。
3. 用户担心权限、隐私或数据边界：
   补发隐私说明页，不把信任问题导向问卷、候补或支付链路。
4. 用户已经完成第一次成功，并明确询问高级工作流方向：
   再发 Pro 路线页，只解释未来高级场景。

## 对外一句话口径

Copylot 的第一目标不是让用户先配置，而是让用户在 30 秒内完成第一次干净复制：先试一个长文、表格或代码块，再把结果直接贴到 AI 或文档里。
