## 3) 页面描述要点（结构化摘录）

以下为对页面长描述的**结构化归纳**（避免临时改字、避免夸大）：

- 痛点：网页复制粘贴“很脏”（广告/导航/表格错位/代码块噪声），用于 ChatGPT/Claude/Gemini 前需要大量清理
- 核心能力：智能区块识别与清理；表格转 Markdown/CSV；代码块清理；Append Mode 多段合并；可选来源归属（标题/URL）
- 隐私口径：本地处理、不收集/不传输复制内容（以页面“Privacy”与隐私政策为准）
