## 用例 C：匿名开关 ON（可复制完整证据包 JSON：CSV + Markdown + rows 互证）

目标：覆盖至少 2 个 campaign + 空 campaign；覆盖 `distCopies=0 -> N/A`；排序稳定；小数格式固定 4 位。

步骤：
1. Options -> 隐私与可观测性：打开“匿名使用数据”开关为 ON。
2. Options -> Pro Tab：设置 `#pro-intent-campaign=twitter`（合法 campaign）。
3. 生成 distCopies（v1-57 口径）：
   - 点击渠道分发工具包任意复制按钮至少 2 次（例如 `#pro-store-url-copy` / `#pro-distribution-pack-copy`），确保记录 `pro_distribution_asset_copied`。
4. 生成 leads（v1-54 口径）：
   - 点击加入候补/问卷相关复制按钮，至少触发 1 次 `pro_waitlist_copied` 或 `pro_waitlist_survey_copied`。
5. 生成第二个 campaign（例如 `xhs`）：
   - 将 `#pro-intent-campaign` 改为 `xhs`，重复步骤 3~4。
6. 生成空 campaign 桶：
   - 清空 `#pro-intent-campaign`（留空），再触发 1 次分发复制或留资复制，使事件 `props.campaign` 缺失/空字符串。
7. 生成 `distCopies=0 -> N/A` 场景：
   - 选择一个 campaign（例如 `ads`），仅触发 leads（步骤 4），不触发分发复制（步骤 3）。
8. 回到 Options -> 隐私与可观测性 -> Pro 面板，点击 `#copy-pro-acquisition-efficiency-by-campaign-evidence-pack`。
9. 将剪贴板 JSON 粘贴到任意文本编辑器中查看。

期望：
- JSON 内字段稳定且可审计：
  - `env` 包含 `windowFrom/windowTo`（7d window，边界一致，可与 v1-58/v1-59 互证）
  - `csv` 为 v1-58 同口径的 7d 合并 CSV（header/字段顺序稳定）
  - `weeklyReportMarkdown` 为 v1-59 同口径的周报 Markdown（标题/表格 header 稳定）
  - `rows` 为结构化行，必须与 `csv` 与 `weeklyReportMarkdown` 表格互证复算一致
- 排序稳定：`leadsPerDistCopy desc -> leads desc -> distCopies desc -> campaign asc`（与 v1-58 rows 一致）。
- `distCopies=0` 的行必须输出 `leadsPerDistCopy=N/A`。
- 小数格式固定 4 位（例如 `0.5000`），便于审计与对比。

截图索引建议：
- `03-privacy-pro-acq-eff-evidence-pack-on-copied.json.png`：匿名 ON 时复制成功通知 + 粘贴 JSON 可见 rows/csv/weeklyReportMarkdown

## 用例 D：互证复算（v1-60 证据包必须与 v1-58/v1-59 同口径且可互证）

目标：证明证据包可审计复核（同一 window 下与 v1-58 合并 CSV、v1-59 周报 Markdown 对齐复算一致）。

步骤：
1. 匿名使用数据 ON。
2. 复制 v1-60：点击 `#copy-pro-acquisition-efficiency-by-campaign-evidence-pack` 得到 evidence pack JSON。
3. 导出 v1-58：点击 `#export-pro-acquisition-efficiency-by-campaign-7d-csv` 下载合并 CSV。
4. 复制 v1-59：点击 `#copy-pro-acquisition-efficiency-by-campaign-weekly-report` 得到周报 Markdown。
5. 核对 v1-60 `env.windowFrom/windowTo` 与 v1-58 CSV / v1-59 Markdown 的 Env 一致（同一时间窗口径）。
6. 按 `campaign` 对齐三份资产，逐行核对：
   - `leads/distCopies/leadsPerDistCopy` 必须一致（分母为 0 时均为 `N/A`）。

期望：
- 任意 campaign 行可互证一致；可复算、可审计；证据包可直接落盘归档。
