# V1-21 口碑入口可观测性对齐：wom_* 埋点补齐 source + 商店 UTM campaign 统一 测试用例

## 目标
验证 V1-21 达到可发布：
- Popup/Options 的 WOM（分享/反馈/去评价/复制分享文案）入口触发时（开关开启）：`copilot_telemetry_events` 中对应 `wom_*` 事件包含 `props.source`，且值与入口一致（`popup`/`options`）
- Popup/Options 打开商店详情页 URL 均携带可审计 UTM：`utm_source=copylot-ext` + 入口对应 `utm_medium` + `utm_campaign=v1-21`
- “匿名使用数据”开关关闭时：不记录、不缓存、不补发；且关闭即清空本地事件
- `bash scripts/test.sh` 全量通过

## 预置条件
- 已在 Chrome/Chromium 加载本地扩展（开发包或生产包均可）
- 能打开 Popup 与 Options 页面：
  - Popup：点击扩展图标打开 Popup（`src/popup/popup.html`）
  - Options：右键扩展 -> 选项，或 `chrome.runtime.openOptionsPage()`（`src/options/options.html`）

## 用例 A（Popup 分享 UTM：打开商店详情页 query 校验）
步骤：
1. 打开 Popup。
2. 点击分享入口 `#share-link`（会打开 Chrome Web Store 详情页）。
3. 检查打开页面的 URL query。

期望：
- URL 的 query 包含：
  - `utm_source=copylot-ext`
  - `utm_medium=popup-entry`
  - `utm_campaign=v1-21`

## 用例 B（Options 分享 UTM：打开商店详情页 query 校验）
步骤：
1. 打开 Options 页面。
2. 点击 WOM 分享入口 `#wom-share-open`（会打开 Chrome Web Store 详情页）。
3. 检查打开页面的 URL query。

期望：
- URL 的 query 包含：
  - `utm_source=copylot-ext`
  - `utm_medium=options-entry`
  - `utm_campaign=v1-21`

## 用例 C（WOM 埋点 source：Popup/Options 来源可审计）
步骤：
1. 打开 Options -> 隐私与可观测性，将“匿名使用数据”开关开启（`#anonymous-usage-data-switch` 置为 on）。
2. 触发 Popup WOM 入口（任选其一）：
   - 点击 `#share-link`（`wom_share_opened`）
   - 点击 `#feedback-link`（`wom_feedback_opened`）
   - 点击 `#rate-link`（`wom_rate_opened`）
   - 点击“复制分享文案”（`wom_share_copied`，需写入剪贴板成功）
3. 触发 Options WOM 入口（任选其一）：
   - 点击 `#wom-share-open`（`wom_share_opened`）
   - 点击 `#wom-feedback-open`（`wom_feedback_opened`）
   - 点击 `#wom-rate-open`（`wom_rate_opened`）
   - 点击 `#wom-share-copy`（`wom_share_copied`，需写入剪贴板成功）
4. 打开 `chrome://extensions` -> 找到本扩展 -> “Service worker” -> Inspect，执行：
   - `await chrome.storage.local.get('copilot_telemetry_events')`
5. 在结果里找到最近的 `wom_*` 事件，检查 `props.source`。

期望：
- Popup 触发的 `wom_*` 事件：`props.source === 'popup'`
- Options 触发的 `wom_*` 事件：`props.source === 'options'`

## 用例 D（开关关闭：不记录 + 关闭即清空 + 不补发）
步骤：
1. 确保已存在至少 1 条 WOM 事件（可先按用例 C 触发一次）。
2. 在 Options 将“匿名使用数据”开关关闭（`#anonymous-usage-data-switch` 置为 off）。
3. 在 Service worker console 执行并记录结果：
   - `await chrome.storage.local.get('copilot_telemetry_events')`
4. 在开关关闭状态下，再次触发任一 WOM 入口（Popup 或 Options 均可）。
5. 再次执行：
   - `await chrome.storage.local.get('copilot_telemetry_events')`

期望：
- 步骤 3：`copilot_telemetry_events` 被清空（不存在该 key 或值为空数组）。
- 步骤 5：仍不应新增 `wom_*` 事件（不记录、不缓存、不补发）。
