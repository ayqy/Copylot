# V1-34 Prompt 作用域一致性修复：悬浮按钮 Prompt 优先选区 测试用例

## 目标
验证子 PRD `prds/v1-34.md` 达到可发布验收口径：

- 悬浮按钮 Prompt：当前块内有“有效选区”时优先作用于选区（表格内选区优先整表；非表格选区优先精确 fragment）
- 悬浮按钮 Prompt：选区不在当前块/无选区时，行为保持既有逻辑（仍作用于当前块）
- 右键菜单 Prompt 行为不变
- 统计与隐私无回归：一次 Prompt 操作仅记录一次 `copy_success` + 一次 `prompt_used`；匿名使用数据关闭时不记录且会清空本地日志

## 用例 A（选区优先：悬浮按钮 Prompt 仅作用于选区）
步骤：
1. 打开任意有多段文本的页面（确保扩展已启用 Magic Copy）。
2. 在同一内容块内划词选中一小段文字（非整段/非整块）。
3. 在该块上通过悬浮按钮打开 Prompt 菜单，点击任一 Prompt。
4. 粘贴剪贴板内容进行观察。

期望：
- 剪贴板内容仅包含选区内容（保留 `<a>/<b>/<code>` 等内联结构对应的文本表达），不应包含未选中的同块其它内容。

步骤：
1. 打开包含表格的页面。
2. 在表格某个单元格内划词产生有效选区（不要框选整表）。
3. 在该表格所在块上通过悬浮按钮打开 Prompt 菜单，点击任一 Prompt。
