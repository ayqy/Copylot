# Copylot 产品路线图与商业化/增长方向

## 现状摘要（以仓库为准）
- 当前版本：`manifest.json` 显示为 `1.1.28`
- 当前状态：已发布到 Chrome Web Store；官网已上线，并开始并行推广获客
  - 官网：`https://copy.useai.online/`
  - 商店：`https://chromewebstore.google.com/detail/ai-copilot-%E2%80%93-magiccopy/ehfglnbhoefcdedpkcdnainiifpflbic`
- 现有核心能力（用户可感知）
  - 智能区块复制（单击/双击触发）与代码块悬停复制
  - 追加模式（Shift 追加合并）
  - 表格一键转 Markdown / CSV
  - Prompt 管理器（`chrome.storage.sync` 同步、内置 Prompt、使用次数统计）
  - Prompt 可配置自动打开 Chat 服务（ChatGPT/Claude/Gemini/Kimi 等内置入口）
- 已知体验缺口（来自 `docs/Future-Features.md` 的方向性记录）
  - “真正的页面去噪/阅读模式级清洗”仍有提升空间
  - Popup 语言切换 UI 缺失（当前跟随系统语言）
  - Prompt 数据（usageCount/lastUsedAt）未充分转化为排序/效率功能
  - 不同入口下 Prompt 应用范围可能不一致，需要统一预期

## 指标口径（Growth Metrics）
- 北极星指标：Activated Active Users（活跃用户中“成功复制 >= 1 次”的人数）
- 核心漏斗：安装 -> 首次打开 Popup -> 首次成功复制 -> 首次使用 Prompt -> 复用
- 关键指标（按优先级）
  - Activation：新装首次会话内首次成功复制率
  - Retention：复用率、每活跃用户复制次数
  - Adoption：Prompt 使用率、表格转换使用率、追加模式使用率
  - Word of Mouth：商店评分、评价率、分享点击率、反馈转化率
  - Revenue：升级入口点击率、候补/留资转化率、付费转化率（后续）、ARPPU（后续）

## 获客与增长抓手（可复用）
- ASO：围绕 “copy to markdown / table to csv / prompt template / ai friendly” 做标题与关键词矩阵；中英文一致化
- 口碑：低打扰评价引导 + 反馈闭环（问题模板化、快速响应、更新日志可感知）
- 内容：围绕核心功能补齐教程与示例，沉淀可复用内容资产
- 裂变：分享入口（带 UTM）+ 可选“复制结果附带 Copylot 来源标记”（默认关闭，避免打扰）

## 商业化策略（方向与边界）
- 定位：Copylot = “AI 时代的复制/采集输入层”；Pro = “更强清洗 + 更强整理 + 更强导出”
- 约束：
  - 默认不收集内容；任何数据采集必须匿名、可关闭、可审计
  - Pro 不依赖读取/上传用户复制内容；优先做本地增强与可选导出
- 路径：
  - 先验证意向（升级入口、候补名单、问卷）-> 再做最小可交付 Pro -> 再扩展到团队与高级能力
