## 用例 A（离线）：问卷为空也可复制（且无隐私越界）

前置：
- 使用仓库内最新 `plugin-*.zip` 解压后以 unpacked 方式加载（禁止用 `src/` 开发态代替验收）。

步骤：
1. 打开 Options -> Pro Tab。
2. 保持“付费意向问卷（可选）”所有字段为空。
3. 点击 `复制问卷（含环境信息）`。
4. 将剪贴板内容粘贴到任意纯文本处（例如本地记事本）。

期望（必须全部满足）：
- 剪贴板内容为 Markdown 文本，包含「隐私提示」与「环境信息」区块。
- 文本不包含网页内容/复制内容/URL/标题（应只出现扩展版本/扩展 ID/语言信息 + 用户手填字段）。
- 问卷字段为空时，仍能稳定生成占位内容（例如 `-`），不报错、不弹窗打扰。

建议取证截图（落盘到 `docs/evidence/v1-48/screenshots/`）：
- `01-options-pro-survey.png`：问卷区块在 Pro Tab 可见且 DOM 稳定（可定位到 `#pro-waitlist-survey`）。

## 用例 B（离线）：部分填写（使用场景 + 选择能力）复制内容断言

步骤：
1. Options -> Pro Tab：在「使用场景」中填写一段短文本（例如 1 行）。
2. 勾选至少 1 个「期望的 Pro 能力」复选框。
3. 点击 `复制问卷（含环境信息）`，粘贴剪贴板内容。

期望：
- Markdown 中「使用场景」区块包含你填写的文本。
- Markdown 中「期望的 Pro 能力」区块包含所勾选能力（口径与 `docs/monetization/pro-scope.md` / `proFeature1~4` 一致）。
- 不包含 URL/标题/网页内容/复制内容。

## 用例 C（离线）：完整填写（含付费意向结构化选项）复制内容断言

步骤：
1. Options -> Pro Tab：填写
   - 使用场景
   - 勾选若干 Pro 能力 + “其他补充”
   - 付费意向：是否愿意付费（Yes/Maybe/No）
   - 可接受价位：按月、按年均选择一个区间
   - 联系方式（可选）：填写邮箱/Telegram 等
2. 点击 `复制并打开候补`：
   - 先检查剪贴板内容（问卷 Markdown）
   - 再确认候补页在新标签打开

期望：
- 剪贴板 Markdown 中包含所有已填写字段（字段缺失即视为失败）。
- “复制并打开候补”会打开候补入口（GitHub `issues/new` 预填模板）。

建议取证截图：
- `02-options-pro-survey-copied.png`：点击复制按钮后按钮反馈为 Copied/已复制（代表写入剪贴板成功）。

## 用例 D（离线）：匿名使用数据 OFF（不记录、不缓存、不补发）

步骤：
1. Options -> 隐私与可观测性：确保“匿名使用数据”开关为 OFF（默认）。
2. Options -> Pro Tab：点击 `复制问卷（含环境信息）` 与 `复制并打开候补` 各至少 1 次。
3. 回到 Options -> 隐私与可观测性：
   - Telemetry Events 面板刷新
   - Pro 意向漏斗摘要/证据包刷新与导出

期望：
- OFF 状态下：不记录、不缓存、不补发。
- 导出内容中不包含 `pro_waitlist_survey_copied`（与既有隐私口径一致）。

## 用例 E（离线）：匿名使用数据 ON（可导出包含新事件的摘要/证据包）

步骤：
1. Options -> 隐私与可观测性：开启“匿名使用数据”开关。
2. Options -> Pro Tab：
   - 点击 `复制问卷（含环境信息）`（触发 `pro_waitlist_survey_copied`）
   - 点击 `加入候补名单`（触发 `pro_waitlist_opened`）
   - 点击 `复制候补文案`（触发 `pro_waitlist_copied`）
3. 回到 Options -> 隐私与可观测性：
   - 刷新 Pro 意向漏斗摘要（应能在 JSON 中看到 `pro_waitlist_survey_copied` 的 `count/lastTs`）
   - 复制「Pro 意向漏斗摘要」与「证据包」并按 `docs/evidence/v1-48/index.md` 命名规范落盘

期望：
- 导出结果可验证 `pro_waitlist_survey_copied` 事件发生过（至少能在 summary 的 `counts/lastTs` 或 evidence pack 的 `events[]` 中找到）。
- 导出结果不包含网页内容/复制内容/URL/标题（仅白名单字段）。

建议取证截图：
- `03-privacy-telemetry-on-pro-funnel.png`：隐私页 Pro Funnel 面板中可见新事件字段（JSON 可见即可）。
