# Copylot Pro 范围与边界（Free vs Pro Planned）

Last Updated: 2026-03-19

本文件用于固化 Copylot 的商业化边界口径，确保可审计、可对外复用、且不暗示 Pro 已上线。

适用范围：本仓库当前版本（以 `manifest.json` 为准）与 Options -> Pro Tab（候补）现有文案口径。

---

## 1) Free（永久免费，不做付费墙）

以下能力为当前已实现的核心体验，并承诺长期免费（不会被“收费墙”阻断）。清单以仓库可审计内容为准（参考：`README.md`、`docs/roadmap.md`、`manifest.json`）。

- 智能区块复制：单击/双击提取正文区域，复制为干净的 Markdown / 纯文本。
- 代码块悬停复制 + 保守清理：保留缩进与空行；在可识别结构下处理行号；保守移除首/末端整行 Copy/复制按钮文案。
- 追加模式（Append Mode）：按住 Shift 复制即可跨页面合并，多段内容用分隔符整理。
- 表格一键转 Markdown / CSV：将网页表格转换为结构化文本，便于粘贴到笔记或表格工具。
- Prompt 管理器：创建/编辑/分类 Prompt 模板；通过右键菜单或 UI 入口一键套用，把“指令 + 内容”拼好复制到剪贴板。
- 工作流增强（已提供的部分）：可配置默认 Chat 服务与“复制后自动打开 Chat”（基于本地设置）。
- 可选附加来源信息：可选在复制内容末尾附上页面标题与 URL（用于溯源，默认由用户在设置中控制）。

> 说明：Free 清单描述的是“已发货能力”，不包含任何候补/规划项。

---

## 2) Pro（规划/候补，不代表已实现）

以下能力仅处于“规划/候补”阶段，用于意向验证与需求收集：**未实现、未上线、不可用**。清单以 Options -> Pro Tab 的 `proFeature*` 口径为准。

| 候选能力（Pro Planned） | 状态 | 备注 |
| --- | --- | --- |
| 高级页面清洗（阅读模式 + 去广告/评论区/推荐区规则） | 规划中 / 未实现 | 目前仅作为方向性描述，不承诺覆盖所有站点 |
| 批量采集与整理（跨页面收集、命名、导出、历史检索） | 规划中 / 未实现 | 不代表已提供“历史库/跨页采集”能力 |
| Prompt Pack 高级库/团队共享（后端能力后置） | 规划中 / 未实现 | 若涉及后端能力，将单独评审隐私与边界 |
| 面向笔记工具的导出能力（Obsidian/Notion 结构化模板） | 规划中 / 未实现 | 当前仅有通用 Markdown/CSV 产出，不承诺专用模板 |

---

## 3) 边界与原则（必须长期成立）

- 不破坏免费核心体验：现有 Free 能力不做付费墙、不降级、不用强打扰弹窗逼迫升级。
- 隐私优先（与 `docs/privacy-policy.md`、`docs/aso/value-prop.md` 一致）：
  - 默认本地处理：提取/清理/格式化在设备本地完成。
  - 不收集/不上传复制内容：不发送复制内容/页面内容到任何 Copylot 服务器。
  - “匿名使用数据”默认关闭且仅本地记录：开启后仅在本地记录匿名事件；不包含复制内容/页面内容/URL/标题；不联网发送；关闭后立即清空本地事件记录。
- 最小权限：仅申请完成核心功能所需权限（以 `manifest.json` 为准）。
- Pro（如未来实现）优先本地增强：任何 Pro 能力优先走“本地可验证、可审计”的增强路径；如需引入网络/后端能力，必须单独立项与隐私评审，且不会默认开启。

---

## 4) 非目标（Non-goals）

本次（V1-29）不做：

- 支付/订阅/登录
- 新增任何数据收集或联网发送
- 新增权限
- 强打扰弹窗或自动弹窗引导
- 任何 Pro 功能本体实现（只做范围/边界固化与口径收敛）
