## 用例 B（教程可复现）
验证目标（`docs/tutorials/table-to-csv-markdown.md`）：
1. 使用稳定可访问示例页面完成一次复制与粘贴验证：
   - 主用：W3Schools - HTML Tables：`https://www.w3schools.com/html/html_tables.asp`
   - 备用：Wikipedia 任意含原生表格页面（教程中给出示例）
2. Table = CSV：
   - Popup 设置：`Enable Magic Copy: ON`、`Table: CSV`、`Extra: OFF`
   - 点击表格任意单元格复制后，粘贴到 Google Sheets / Excel / Numbers 应能自动分列成表格
3. Table = Markdown：
   - Popup 设置：`Table: MD`、`Extra: OFF`
   - 点击表格复制后，粘贴到 GitHub / Obsidian / Notion（或其他支持 Markdown 的编辑器）应能以 Markdown 表格形式呈现
4. 教程强调“3 分钟内完成首次成功”的最短路径，并且术语/开关命名与 Popup UI 一致（参考 `src/popup/popup.html`）。

- 结果：通过
- 备注：
  - 使用 W3Schools 示例表格验证：CSV 粘贴到表格工具后自动分列；Markdown 粘贴到 Markdown 预览环境可渲染为表格。
  - 已在教程中明确建议关闭 `Extra`（避免在 CSV/表格后追加来源信息影响导入）。
