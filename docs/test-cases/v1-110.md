## 用例 A：Options Pro 分享工具统一为安装 / 隐私 / 路线 / 分享

步骤：
1. 打开 `src/options/options.html#pro`。
2. 填写 `campaign=twitter`。
3. 依次触发官网路线页、分享文案、商店安装链接、完整投放包复制。

期望：
- `#pro-waitlist-button` 打开的仍是 `/pricing`。
- “复制分享文案”产物包含商店安装链接、隐私说明页和 Pro 路线页。
- “复制完整投放包”产物包含官网首页、商店安装页、隐私说明页和 Pro 路线页四类固定链接。
- 复制内容不包含 `survey`、`问卷`、`候补名单` 等公开收集口径。
