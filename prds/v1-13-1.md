# V1-13-1 Options Pro Tab 新增“口碑与支持”区块（稳定 DOM + i18n）

## 目标
在不影响现有 Pro Tab（候补）功能与样式的前提下，在 Options -> `#pro-tab` 补齐“口碑与支持”区块，提供可 i18n 且可审计的稳定 DOM 标识，供后续回归与自动化使用。

## 具体任务
1. 在 `src/options/options.html` 的 `#pro-tab` 新增区块容器：
   - 容器：`#wom-actions-panel`
   - 按钮（用户点击触发，不自动弹窗/自动复制）：
     - `#wom-share-open`
     - `#wom-share-copy`
     - `#wom-rate-open`
     - `#wom-feedback-open`
2. i18n：
   - 复用既有 key：`shareWithFriends` / `copyShareText` / `rateUs` / `feedback`
   - 新增区块标题/说明 key：`womActionsTitle` / `womActionsHint`
   - 补齐 `_locales/en/messages.json` 与 `_locales/zh/messages.json`

## 验收标准
- Options Pro Tab 存在 `#wom-actions-panel`，包含 4 个入口按钮且均可点击
- DOM 标识稳定、可用于用例与未来自动化
- 文案可 i18n（中英文均存在）
- 不新增权限、不新增第三方依赖、不影响既有 Pro Tab 候补功能
