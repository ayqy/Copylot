# v1-111 截图顺序

## 主标题

- `30 秒完成第一次干净复制`

## 前三个卖点

1. 长文：去掉导航、广告和杂项说明，直接得到干净 Markdown / 纯文本。
2. 表格：一键转成 Markdown / CSV，粘贴后结构仍然清楚。
3. 代码块：保留缩进与空行，尽量去掉行号和复制按钮噪声。

| 顺序 | 截图对象 | 画面重点 | 对齐要求 |
| --- | --- | --- | --- |
| 1 | Popup 首屏 | 主标题 + 三步指引 + 长文/表格/代码块卖点 | 与 Popup 首屏、官网 Hero 完全同口径 |
| 2 | 长文复制结果 | 去导航、去广告后的干净 Markdown | 证明“第一次复制出来就能用” |
| 3 | 表格复制结果 | Markdown / CSV 双形态 | 保持与商店前三个卖点顺序一致 |
| 4 | 代码块复制结果 | 保留缩进、减少噪声 | 对齐代码场景价值证明 |
| 5 | 粘贴结果 | 将第一次结果粘贴到 ChatGPT、Claude 或文档 | 明确“复制后贴到哪里” |
