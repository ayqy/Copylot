# v1-108 测试用例

## 范围

- 回退 Popup “你最想先解决什么”功能
- 扩展内完全去掉表单、问卷、候补显性路径
- 调整增长与官网口径为浏览器插件增长闭环优先

## 用例 A：Popup 不再展示问卷或候补入口

步骤：
1. 打开插件 Popup。
2. 观察 Pro 相关区域。

期望：
- 不存在“你最想先解决什么”意向卡
- 不存在问卷入口
- 不存在候补入口
- 只保留 `#upgrade-pro-entry`

## 用例 B：Popup 的 Pro 入口只打开扩展内路线页

步骤：
1. 在 Popup 点击 `#upgrade-pro-entry`。
2. 观察打开的新页面。

期望：
- 打开的页面为 `src/options/options.html#pro`
- 页面中存在 Pro 路线说明和分享工具
- 页面中不存在表单、问卷、候补填写区

## 用例 C：Options Pro 页只保留路线与分享工具

步骤：
1. 打开 `src/options/options.html#pro`。
2. 观察 Pro 区块。

期望：
- 存在 Free / Pro 路线说明
- 存在来源标记输入框
- 存在官网路线页、商店安装链接、分享文案、完整投放包四类分享动作
- 不存在问卷表单、联系方式输入、复制问卷按钮

## 用例 D：官网承接页改为安装闭环优先

步骤：
1. 打开首页。
2. 打开 `/pricing`。
3. 打开 `/privacy`。

期望：
- 首页主 CTA 以安装和上手为主
- `/pricing` 为 Pro 路线说明页，不是支付模板
- `/privacy` 可访问，且解释本地处理与匿名开关

## 用例 E：分享资产全部围绕增长闭环

步骤：
1. 在 Options -> Pro 填写 `campaign`
2. 依次点击复制官网路线页、复制商店安装链接、复制分享文案、复制完整投放包

期望：
- 所有复制内容都不包含表单、问卷、候补引导
- 分享文案主目标是安装、使用、推荐
- 投放包包含官网首页、商店安装页、Pro 路线页三类链接
