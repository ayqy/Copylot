## 用例 A：强制触发 Popup “1 分钟问卷”提示 -> 跳转问卷区（source=popup）

前置说明：
- `shouldShowProPrompt` 受安装时长/成功复制次数/Prompt 使用等门槛影响，为避免等待，可通过写入 `chrome.storage.local` 的 `copilot_growth_stats` 强制满足门槛。

步骤：
1. 在真实 Chrome 环境安装当前插件版本（以 `manifest.json` 为准）。
2. 打开 Popup（任意方式），打开 DevTools Console（Popup 页面）。
3. 写入增长统计以满足门槛（示例，字段可按需补齐）：
   - `const now = Date.now();`
   - `chrome.storage.local.set({ copilot_growth_stats: { installedAt: now - 14 * 24 * 60 * 60 * 1000, successfulCopyCount: 999, firstPromptUsedAt: now - 1_000, proPromptShownCount: 0 } })`
4. 关闭并重新打开 Popup。
5. 确认出现“1 分钟问卷”提示，点击主按钮（Join/填写问卷）。

期望：
- Popup 显示低打扰提示（可“稍后/不再提示”）。
- 点击主按钮后打开 Options 新标签页，URL 符合：
  - `src/options/options.html?pro_survey_source=popup#pro-waitlist-survey`

## 用例 B：匿名开关 OFF -> 问卷区提示“不计入样本” + 一键开启

步骤：
1. 打开 Options，进入「隐私与可观测性」页，关闭「匿名使用数据」开关（关闭会清空本地匿名事件日志）。
2. 进入「Copylot Pro（候补）」页，滚动到 `#pro-waitlist-survey`。
3. 观察问卷区的“计入样本”提示与按钮。
4. 点击“一键开启并计入样本”按钮。

期望：
- 开关 OFF 时：
  - 问卷区出现提示：“未开启匿名使用数据：本次填写/复制不会计入 `survey_intent`…”
  - 存在按钮：“开启并计入样本”
- 点击按钮后：
  - `isAnonymousUsageDataEnabled` 变为 ON（可切换到隐私页验证开关已勾选）
  - 问卷区提示与按钮隐藏（避免噪音）

## 用例 C：匿名开关 ON -> 产生样本 + 导出证据包并复核口径

步骤：
1. 保持匿名开关 ON（可用用例 B 一键开启）。
2. 在 `#pro-waitlist-survey` 填写任意字段（可为空），点击「复制问卷（含环境信息）」或「复制并打开候补」。
3. 在 Options -> 隐私与可观测性：
   - 展开「Pro 意向漏斗摘要」面板，确认漏斗字段存在
   - 点击「下载 Pro 意向跑数证据包（JSON）」并下载文件
4. 打开导出的 JSON，复核关键字段与隐私红线。

期望：
- UI 侧可观察：
  - `pro_waitlist_survey_copied >= 1`（若从 Popup 跳转进入问卷，`source=popup` 可被观察）
  - `survey_copied_per_prompt_shown` 存在（分母为 0 时为 `null`）
- 证据包侧可复核（v1-90 导出）：
  - `proFunnelSummary.*.rates.survey_copied_per_prompt_shown` 存在
  - `proWaitlistSurveyIntentDistribution.survey_intent >= 1`
- 隐私红线：证据包不包含 URL/标题/网页内容/复制内容/联系方式明文。
