# 问题修复和功能优化（V1-92 Pro 意向样本量提升：Popup 问卷提示升级 + 一键开启匿名使用数据 + 漏斗证据口径固化）

## 目标
在不引入支付/订阅、不新增权限、不上传用户复制内容的前提下，把「Pro 候补问卷」的真实可量化样本量（`survey_intent = count(pro_waitlist_survey_copied)`）继续推高：让更多满足门槛的用户在 Popup 里看到“1 分钟问卷”低打扰提示，并在问卷区通过“计入样本”提示 + 一键开启匿名使用数据（可随时关闭且关闭会清空）把意向行为稳定记入漏斗与证据包，为 S4「收款/订阅链路打通」提供可审计、可复盘的 go/no-go 决策输入。

## ULTRA IMPORTANT 第一原则：任务开始后，不要停下来，直到完成目标
**所有细节，都由你来定，判断依据是根据核心目标决定应该如何做，而不要停下来问我**

## 具体任务
1、Popup 侧“Pro 候补提示”统一为“1 分钟问卷”引导（提升样本触达）
  - 修改：`src/popup/popup.html`、`src/popup/popup.ts`、`_locales/*/messages.json`
  - 行为要求（低打扰、可关闭）：
    - 仍复用现有触发门槛（`shouldShowProPrompt`），不新增强打断弹窗
    - 提示文案与按钮文案统一为“1 分钟问卷”价值解释（用于决定是否做订阅/做哪些 Pro 能力）
    - 主按钮点击后：打开 Options 并定位到问卷区（必须带来源归因）：
      - `src/options/options.html?pro_survey_source=popup#pro-waitlist-survey`
    - 「稍后/不再提示」逻辑保持不变（仍可 snooze/永久关闭）
  - 口径要求（可复盘）：
    - Popup 出现提示：`pro_prompt_shown`（已存在）
    - 点击主按钮：`pro_prompt_action`（已存在；action 仍复用 `join|later|never`，不新增枚举）

2、问卷区“计入样本”提示 + 一键开启匿名使用数据（提升样本可计入率）
  - 修改：`src/options/options.html`、`src/options/options.ts`、`_locales/*/messages.json`
  - 交互要求（透明、可逆、零打扰）：
    - 当「匿名使用数据」开关为 OFF 时，在 `#pro-waitlist-survey` 显示一条提示：
      - “未开启匿名使用数据：本次填写/复制不会计入 `survey_intent` 统计（仍可继续填写）”
    - 提示中提供 1 个按钮（i18n 文案）：
      - “开启并计入样本”（点击后仅修改本地设置为 ON，并立刻刷新相关面板；用户可随时在隐私页关闭且关闭会清空本地日志）
    - 当开关为 ON 时隐藏该提示与按钮（避免噪音）
  - 隐私与门禁：
    - 不得新增权限；不得联网发送；不得记录/导出 URL/标题/网页内容/复制内容/联系方式明文
    - 仅复用现有匿名开关逻辑：OFF 时不记录/不缓存/不补发，且会清空本地匿名事件日志

3、漏斗口径固化 + 可导出证据（让“样本量提升”可被验证）
  - 修改：`src/shared/pro-funnel.ts`、`src/options/options.ts`、`scripts/unit-tests.ts`
  - 漏斗口径补齐（稳定字段）：
    - 在 `ProFunnelSourceStats.rates` 新增 1 个稳定字段：
      - `survey_copied_per_prompt_shown = pro_waitlist_survey_copied / pro_prompt_shown`
    - 在 Options -> 隐私与可观测性 ->「Pro 意向漏斗摘要」与 v1-90 的证据包导出中必须可观察该字段
    - 分母为 0 时该字段必须为 `null`（避免 NaN/Infinity 与口径误读）
  - 自动化门禁（必须可执行）：
    - 更新/新增单测：`scripts/unit-tests.ts`
      - 断言 `survey_copied_per_prompt_shown` 字段存在、类型稳定、且在分母为 0 时为 `null`
    - `bash scripts/test.sh` 全量 PASS
  - 新增测试用例文档（固定路径）：`docs/test-cases/v1-92.md`
    - 必须覆盖（离线可复现，不等待 48h）：
      - 触发 Popup 提示 -> 点击主按钮 -> Options 自动定位到 `#pro-waitlist-survey`（来源归因 `popup`）
      - 匿名开关 OFF/ON 两种情况下的“计入样本”提示与一键开启行为
      - 产生至少 1 条 `pro_waitlist_survey_copied` 后导出 v1-90 证据包，并验证 `survey_intent >= 1` 且包含新漏斗字段
    - 必须包含一次回归记录（PASS/BLOCKED + 导出文件名/证据文件名/截图索引）
